CSS Properties Cheat Sheet

Color

Property Name Description
color Color

Sets the color of the text for the selected element.

The color can be set in a variety of ways including:

  • Named colors like 'green' or 'red'
  • Hexadecimal (#000000)
  • rgb(xx,yy,zz)
  • rgba(xx,yy,zz,0.5)
  • hsl(xx,yy%,zz%)
  • hsla(xx,yy%,zz%,0.5)
background-color Background Color Sets the background color of the specified element.
Can be set in the same ways as mentioned above for the color property.

Typography

Property Name Description
font-family Font Used to change the font of an element.
Can be a web safe font or can link in a web font with an @font-face ruleset.
font-weight Font weight (Bold or Light) Used to bold text or make it lighter.
font-style Italics Used to italicize text.
text-transform Uppercase or Lowercase Used to make text all uppercase or all lowercase.
letter-spacing Letter Spacing Used to set a spacing between letters.
word-spacing Word Spacing Used to set spacing in between words.
line-height Line Height Sets how tall each line containing text should be.
text-align Text Alignment Aligns text to some side of the container.
